## Hapi Simple API


### Dependencies

- `hapi joi boom mongoose`

### Usage

```
npm install
npm start
```


### API

- `GET     /players`        : list all players
- `POST    /players`        : create new player
- `GET     /players/{name}` : get player by name
- `PUT     /players/{name}` : update player by name
- `DELETE  /players/{name}` : delete player
